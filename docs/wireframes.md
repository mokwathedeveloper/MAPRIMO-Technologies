# System Hierarchy & Layout

## Marketing Hierarchy
- **Header:** Sticky glassmorphism navigation.
- **Hero:** High-impact typography with primary CTA.
- **Social Proof:** Infinite scrolling or static grid of partner logos.
- **Dynamic Content:** 
  - Case Study Grid (app/work)
  - Blog Feed (app/blog)
  - Podcast Player (app/podcast)

## Admin Dashboard Layout
- **Sidebar:** Navigation links (Projects, Leads, Blog, etc.).
- **Header:** Stats summary (Count of active leads/projects).
- **Table/List View:** Searchable/Sortable content list.
- **Editor:** Form-based CRUD with optimistic UI updates.
